HTML Forms exercises

Build a signup form

  1. Create a new folder on your computer. Call it 'form-exercise'.
  2. Save the starter HTML file to the form-exercise folder by right-clicking this link and selecting Save Link As.
  3. Do the same with the css.
  4. Create a new file in the form-exercise folder called 'form.js'.
  5. Create a listener for click events on the #signup link that will toggle the #ss-form form.
  6. Ensure that the link is not being followed (i.e. remove the # from the URL)
  7. Create a variable called submitted and set it to false.
  8. Create a listener for submit events on the #ss-form form that will set the submitted variable to true.
  9. Create a listener for load events on the #hidden-iframe that, if submitted is true will:
    • sets the text on div#message to "Signed up successfully"
    • fades in div#message
    • fades out div#message over 5 seconds
    • hide the form
